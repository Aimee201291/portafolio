SmartCAD Web
Previous Next

Desarrollo de aplicación web que permite llevar a cabo la gestión de incidentes captados por cámaras de seguridad. SmartCAD Web permite consultar alertas obtenidas a partir de eventos detectados por cámaras de seguridad, reproducir grabaciones de video de alertas asociadas a eventos mientras se muestran cuadros delimitadores alrededor de las personas que generaron dichas alertas, seleccionar alertas generadas para registrar incidentes que serán atendidos por los departamentos correspondientes (policías, bomberos, ambulancias, etc.), y descartar aquellas alertas generadas que no corresponden a incidentes. Para este proyecto fue necesario integrar un software de gestión de video (Milestone Xprotect Professional), un software de análisis de video (savVi de Agent Vi), y un software de reconocimiento automático de placas de vehículos (VaxALR de Vaxtor Systems).
